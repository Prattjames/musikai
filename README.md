# musikai

Artificial Intelligence music recommendation. Context based machine learning applied in the music field.

---

### Features

- Cross-platform music player
- Music Recommendation based on machine learning
- Clean and polished
- Playlists
- Queue management
- Shuffle, loop
- Covers
- Dark theme
- Playback speed control
- Sleep mode blocker
- Minimize to tray
- Supported formats:
    - mp3
    - mp4
    - m4a/aac
    - flac
    - wav
    - ogg
    - 3gpp
